# PlugHub_Project_Server

팀 프로젝트인 PlugHub 프로젝트를 진행하면서 만들었던 로그인 서버와 DB 서버입니다.<br>
처음 해본 백엔드 작업이라 부족한 점은 많았지만 많은 경험을 쌓을 수 있었습니다.<br><br>
**사용 프로그램은 NodeJS 이며, 요청을 위한 라이브러리는 Retrofit을 사용했습니다.**
